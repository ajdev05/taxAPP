Tax Calculator API Documentation

Endpoint

The API endpoint for calculating tax is:

GET /api?taxableIncome={your_income}&filing={your_tax_filing}

Parameters

The following parameters are accepted

Parameter Description
taxableIncome Your income amount for tax calculation.
filing Your tax filing status (See the documentation below for your code).

Filing Status Codes

Tax filing status codes and their descriptions

Value Description
0 Single
1 Married jointly or qualifying widow(er)
2 Married separately
3 Head of household


Usage Example

Example Usage for the API:

GET /api?taxableIncome=50000&filing=0
